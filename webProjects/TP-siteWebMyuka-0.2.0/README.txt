J'ai tenu à développer seulement un site web semi-dynamique (HTML, CSS/SASS, JS) pour le projt MYUKA au lieu de créer 2 pages web statiques (HTML, CSS).

Voici l'architecture du dossier :
- dev : dossier "développement" pour l'HTML.
- js : dossier pour les fichiers JavaScript.
- public :
         * css : dossier du CSS compilé.
         * images : dossier des images utilisées pour le site.
- sass : dossier contenant le pattern 7-1 pour l'architecture des sous-dossiers .scss.

Le site Web est full responsive. Il y un humburger menu pour le responsive dans le header. Egalement il y a dans le footer un Dark Mode si tu cliques sur le switch button.
